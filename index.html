Privacy Policy

Effective date: March 28, 2026

Glowdrop is a simple hydration companion designed to help users build healthy water drinking habits. Your privacy is a top priority.

Data Collection

Glowdrop does not collect, store, or transmit any personal data to external servers.

Local Storage

All user data is stored locally in your browser using Chrome storage. This includes:

  • Daily water goal
  • Reminder interval
  • Active hours
  • Hydration progress

This data never leaves your device.

Permissions Used

  • Storage: Save preferences locally
  • Alarms: Schedule reminders
  • Notifications: Notify the user

Third-Party Services

Glowdrop does not use:

  • External APIs
  • Analytics tools
  • Tracking systems
  • Advertising services

Personal Data

Glowdrop does not collect:

  • Name or identity information
  • Email address
  • Location data
  • Browsing history
  • Health or financial data
