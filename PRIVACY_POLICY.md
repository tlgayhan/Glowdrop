# Privacy Policy for Glowdrop – Water Reminder & Hydration Companion

Last updated: 2026-03-28

Glowdrop is a Chrome extension that helps users build a healthier water-drinking habit with reminders and a character-based progress experience.

## Summary

Glowdrop does not sell your data.
Glowdrop does not use advertising trackers.
Glowdrop does not send your hydration data to an external server.

The extension stores hydration-related settings and progress locally in your browser so the product can function correctly.

## Information Glowdrop Stores

Glowdrop stores the following data locally using `chrome.storage.local`:

- Daily water count
- Daily goal
- Streak
- Last drink timestamp
- Reminder interval
- Snooze interval
- Active hours
- Theme preference
- Notification state
- Recent daily history used for statistics

This data stays on the device where the extension is installed unless you export, sync, or share it yourself through browser features outside Glowdrop.

## Permissions Used

Glowdrop uses only the following Chrome permissions:

- `storage`
  - Used to save hydration progress, settings, theme, and recent statistics locally
- `alarms`
  - Used to schedule water reminders and snooze reminders
- `notifications`
  - Used to show hydration reminders

Glowdrop does not request host permissions, page access, browsing history, cookies, or content-script access.

## Data Sharing

Glowdrop does not transmit personal data to a remote server.
Glowdrop does not share personal data with third parties.
Glowdrop does not sell personal data.

## Analytics and Tracking

Glowdrop does not include third-party analytics, advertising SDKs, or tracking pixels.

## Children

Glowdrop is a general productivity and wellness tool and is not directed to children under 13.

## Changes to This Policy

If Glowdrop's data practices change in the future, this privacy policy should be updated before those changes are released.
